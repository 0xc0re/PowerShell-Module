Welcome to the Rubrik PowerShell Module
========================

This is a community project that provides a Windows PowerShell module for managing and monitoring Rubrik's Cloud Data Management fabric by way of published RESTful APIs. If you're looking to perform interactive automation, setting up scheduled tasks, leverage an orchestration engine, or need ad-hoc operations, this module is intended to be valuable to your needs. Below is a quick YouTube video that explains how to begin using the module.

The code is open source, and `available on GitHub`_.

.. _available on GitHub: https://github.com/rubrikinc/PowerShell-Module
